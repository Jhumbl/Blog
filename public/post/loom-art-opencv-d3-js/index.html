Loom Art - Image Processing - OpenCV/D3.js

Jack Humble · 2019/04/27 · 1 minute read

The inspiration for this project came from two places. The first after being unable to solve this puzzle in the New Scientist (which can be compared to drawing lines through the pixels of an image). The second was seeing other examples of this type of art style, for example this piece by Petros Vrellis, and wondering what method was used to determine the order in which the threads should be placed. Below is my best guess at how the algorithm works. Press the play buttons to see the looms build up thread by thread.

D3.js and Python(OpenCV)

This visualisation was created with D3.js (Data Driven Documents), a javascript library allowing developers to bind data to elements of the document object model and draw svg elements whose attributes are determined by the underlying data. More information can be found at https://d3js.org/. Python and the cv2 library were used to implement the algorithm. The code for this project can be found on github here.
